# folderer
scans celected directory, makes folders named by the file extenctions and sorts the files into them
Dependencies:
    from source: shutil, os, tkinter, pathlib, python
    from binary: none, everything compiled in
Requirements:
    tested on Xubuntu 24.04 LTS with python 3.12.1
    may work on other python version (never tested, whould be glad if reported)
    should work on most linux distros (never tested, whould be glad if reported)
